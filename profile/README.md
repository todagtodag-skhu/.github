## 토닥토닥 - 칭찬스티커판 기반 습관 형성 도우미 서비스

---

## ✍️ 프로젝트 개요

- **프로젝트팀명:** 토닥이와 성장이
- **프로젝트 기간:** 2026.03.20 ~ 2026.04.09
- **프로젝트 형태:** 경진대회 참가 및 서비스 출시, 운영경험 목적
- **목표:** 기존 습관 서비스들의 한계를 극복하고, 아동 및 고령층을 대상으로 미션형 습관 형성 및 복약지도를 위해 활용할 수 있는 서비스를 제공하는 React Native 기반 어플리케이션 서비스 개발
- **주요 타겟 사용자:** 습관 형성 또는 교정이 필요한 5세 - 19세 미만의 어린이, 학생, 복약지도가 필요한 65세 이상 고령층의 사용자
