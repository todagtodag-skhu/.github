## ✍️ 프로젝트 개요

- **프로젝트팀명:** 토닥이와 성장이
- **프로젝트 기간:** 2026.03.20 ~ 2026.04.09
- **프로젝트 형태:** 경진대회 참가 및 서비스 운영 경험을 목표로 한, 팀 프로젝트
- **프로젝트 목적:** 기존 습관 형성 서비스의 한계를 보완하고, 부모-자녀 / 보호자-관리 대상 관계에서 활용 가능한 **칭찬 스티커판 기반 습관 형성 서비스**를 구현
- **프로젝트 소개:** 토닥토닥은 칭찬 스티커판 기반의 미션형 습관 형성 서비스입니다. 보호자와 아이가 함께 목표를 만들고, 미션 수행과 칭찬, 보상 경험을 통해 올바른 생활 습관 형성을 돕습니다.
- **주요 타겟 사용자:** 습관 형성 또는 교정이 필요한 아동·청소년, 부모-자녀, 조부모-손주 스승-제자 등 보호자-관리 대상 관계의 사용자

---

## 🚀 프로젝트 목표

- 스마트폰 과의존과 자기조절의 어려움을 겪는 아동·청소년에게 규칙적인 생활 습관 형성 경험을 제공합니다.
- 반복적인 루틴, 보호자의 피드백, 보상 구조를 통해 스스로 실천하기 어려운 사용자의 습관 형성을 돕습니다.
- 기존 습관 앱의 개인 중심 구조를 탈피하고, 보호자와 사용자가 함께 만드는 상호작용 기반 습관 형성 환경을 제공합니다.
- 추후에는 복약 지도, 스크린타임 관리 등으로 확장 가능한 서비스 기반을 마련하는 것을 목표로 합니다.

---

## 🔍 문제 인식

기존의 습관 형성 서비스는 주로 개인이 스스로 목표를 세우고 실천 여부를 기록하는 구조에 집중되어 있습니다.  
하지만 아동이나 고령층처럼 보호자 또는 가족의 개입이 중요한 사용자에게는  
단순 기록형 서비스만으로 지속적인 습관 형성을 기대하기 어렵습니다.

또한 기존 서비스는 다음과 같은 한계를 가집니다.

- 개인 중심의 기록 기능에 치우쳐 있음
- 부모/보호자가 직접 미션을 부여하고 관리하는 기능이 부족함
- 칭찬 스티커판을 채워가는 **시각적 재미와 성취감** 제공이 약함
- 부모-자녀, 보호자-대상자 간의 **관계 기반 상호작용 구조**가 부족함

토닥토닥은 이러한 문제를 해결하기 위해  
**칭찬 스티커판 기반의 관계형 습관 형성 서비스**를 목표로 개발되었습니다.

## 📌 주요 기능

### 1. 역할 기반 사용자 구조
- 사용자 역할을 **토닥이**와 **성장이**로 구분
- 가입 시 역할 선택 가능
- 초대코드를 통해 손쉽게 상호 연결 가능
- 1명의 토닥이가 여러 명의 성장이를 관리할 수 있는 구조 지원

### 2. 미션 기반 습관 형성
- 토닥이가 성장이에게 생활 습관, 인성 함양, 할 일 등의 미션 부여
- 성장이가 미션 수행 후 완료 요청 가능
- 토닥이가 확인 후 칭찬 스티커 지급

### 3. 칭찬 스티커판 시스템
- 지급받은 스티커를 직접 스티커판에 부착 가능
- 스티커판을 채워가는 재미와 시각적 만족감 제공
- 목표 달성 시 보상과 연결하여 지속적인 참여 유도

### 4. 보상 기반 동기부여
- 스티커판 완성 시 토닥이가 설정한 보상 제공
- 단순한 할 일 완료를 넘어 **칭찬과 성취 경험** 강화

### 5. 확장 가능성
- 아동 대상 습관 형성 외에도
  - 고령층 복약 지도
  - 안부 확인
  - 생활 루틴 관리
  등으로 서비스 확장 가능
